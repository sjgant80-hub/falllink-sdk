# falllink-sdk

**FallLink · reusable P2P library for the estate** — sovereign SDK extracted from `falllink/index.html`.

## Install
```bash
npm install @ai-native-solutions/falllink-sdk
```

## Use
```js
import * as falllink from '@ai-native-solutions/falllink-sdk';
```

- Playground: https://sjgant80-hub.github.io/falllink-sdk/
- MCP wrapper: [`@ai-native-solutions/falllink-mcp`](https://github.com/sjgant80-hub/falllink-mcp)
- HTTP API: [`@ai-native-solutions/falllink-api`](https://github.com/sjgant80-hub/falllink-api)
